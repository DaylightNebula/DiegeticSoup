# Diegetic Soup
This is a project to add support for digetic UI to Minestom and Paper Minecraft servers.
Diegetic UIs allow for a more modern and intuitive user experience with UIs placed directly in the game world.
This also allows us to get around the UI limitations that are traditionally associated with Minecraft servers.

## This Projects State
Currently, this project is very experimental and mostly serves as a place to test and prototype ideas instead
of producing a finalized product.  Therefore, use at your own risk, and all contributions are welcome and 
appreciated.  Documentation for this project is also a bit sparse, but I am trying to make documentation as I
finalize components that I am happy with.
